# 打答 - 你的智能记分小助手 🎲

欢迎使用 **打答**！这是一个专为打答设计的记分和计算工具，能帮你轻松记录每一局的分数，自动计算输赢，彻底告别繁琐的手工计算。

---

## 🚀 快速上手

### 第一步：进入房间

打开应用后，你有两个选择：

1. **创建房间**：如果你是发起人，点击“创建房间”，输入一个数字作为房间号（例如 `888`）。
2. **加入房间**：如果朋友已经建好了房间，点击“加入房间”，输入即使的房间号即可同步数据。

### 第二步：设置玩家

如果是新房间，系统会提示你输入 **4位玩家的名字**。

* 输入名字，中间用空格或逗号隔开（例如：`张三 李四 王五 赵六`）。
* 点击确认，就准备好啦！

---

## 📝 记分指南

### 1. 开始记分

在表格的 **“全部”** 行下方，就是当前的记分区域：

* **单人记分**：直接点击某位玩家下方的格子或者名字，输入分数。
* **整局记分**：点击行末尾的 **“记分”** 按钮，效率更高！

### 2. 开启新场次

默认分数达到100分后，系统会自动开启新场次。如果想重新开始一轮累计：

* 点击表头的 **“加场”** 按钮。
* ⚠️ **注意**：开启新场次后，旧的场次会被锁定，不能再修改哦。

### 3. 修改每分金额

默认每分 `0.3` 元。如果你们玩的大小不一样：

* 点击顶部的 **“修改”** 按钮，输入新的金额即可。

---

## 💰 自动结算

当游戏结束，或者你想看现在的输赢情况时：

1. 点击 **“全部”** 行右侧的 **“计算”** 按钮。
2. 系统会自动根据当前的积分和每分金额，算出每个人的输赢金额。
3. **负数**代表输了，**正数**代表赢了。
4. 页面还会显示你们玩了多久（活动时长）。

---

## 💡 常见问题 (Q&A)

**Q: 记错了分怎么办？**
A: 只要是当前场次的分数，点击对应的格子或“修改”按钮就能改。如果已经“加场”锁定了，就改不了咯。

**Q: 怎么把数据分享给朋友？**
A: 告诉他们房间号，让他们在自己的手机上“加入房间”，数据是实时同步的！

**Q: 怎么退出房间？**
A: 点击右上角的“退出房间”按钮即可回到主页。
